# Climate-Risk
Data visualisation project on how Climate Risk affects Mortgage industry in dash python.

Climate risk was divided in two main risks as Physical Risk and Transition Risk. All project was realised in Python (Pandas, SciKit, Dash, Plotly), Html/CSS.

Dataset - DF.xlsx (all data are fake)
Climate Risk.ipynb - Jupyter Notebook file with all quantification, formulas and dash graphs programming
Development.py - All dashboard code uploaded in single python file
